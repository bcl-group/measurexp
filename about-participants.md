# 被験者ファイルについて

- 親ディレクトリは "%Y%m" 形式
- 子ディレクトリは被験者とし A, B, ...
- 子の下にデータファイルをおく
    - 筋電位データは EMG
    - 床反力データは FPL
    - 軌道データは MCP

## セッション番号
セッション番号統一のため、3 桁で管理する。
- 百の位: 条件の種類
- 十の位: 何回目の条件か
- 一の位: 条件の順番

**失敗したセッションは入れない**


> 例:
```
.
├─202103
│  ├─A
│  │  ├─EMG-101.csv
│  │  ├─FPL-101.csv
│  │  ├─MCP-101.csv
```
